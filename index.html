Informasi Tentang Debian Linux

Penjelasan lengkap mengenai Debian GNU/Linux, mulai dari deskripsi hingga sejarah perkembangannya

Apa itu Debian?

Debian adalah sistem operasi berbasis Linux yang sepenuhnya gratis dan open source. Dikenal sebagai "Sistem Operasi Universal" karena dapat digunakan pada berbagai perangkat keras, mulai dari komputer desktop, server, hingga perangkat embedded.

Debian menggunakan kernel Linux dan sebagian besar utilitas dasar sistem berasal dari proyek GNU, sehingga dinamakan Debian GNU/Linux. Sistem ini memiliki lebih dari 59.000 paket perangkat lunak yang siap diinstal.

Sejarah Debian

1993

Debian didirikan oleh Ian Murdock pada 16 Agustus 1993. Nama "Debian" berasal dari kombinasi nama Ian dan istrinya Debra.

1996

Debian 1.1 "Buzz" dirilis sebagai versi stabil pertama. Setiap rilis diberi nama karakter dari film Toy Story.

2004

Menjadi dasar bagi distribusi populer seperti Ubuntu, Linux Mint, dan Kali Linux.

2024

Debian 12 "Bookworm" dirilis dengan pembaruan dan fitur-fitur modern.

Cara Instalasi Debian Linux

Panduan lengkap instalasi Debian dari persiapan hingga selesai

Persiapan Sebelum Instalasi

Pastikan semua kebutuhan terpenuhi sebelum memulai instalasi

1

Download ISO Debian

Download file ISO Debian dari situs resmi. Pilih Netinst (~400MB) untuk instalasi via internet atau Complete DVD untuk instalasi offline.

2

Siapkan USB Drive

USB Flash Drive minimal 4GB atau DVD kosong untuk membuat media instalasi bootable.

3

Tool Bootable USB

Download aplikasi seperti Rufus (Windows), Etcher (Multi-platform), atau gunakan dd command di Linux.

4

Backup Data

Backup semua data penting dari komputer Anda untuk menghindari kehilangan data saat instalasi.

5

Cek Spesifikasi

RAM: Min 512MB (Ideal 2GB+) ‚Ä¢ Storage: Min 10GB (Ideal 20GB+) ‚Ä¢ Processor: Pentium 4 atau lebih baru

6

Siapkan Koneksi Internet

Untuk instalasi Netinst, pastikan koneksi internet stabil agar proses download paket berjalan lancar.

Langkah-langkah Instalasi

01
Persiapan Media

Buat Bootable USB

Gunakan Rufus atau Etcher untuk membuat USB bootable dari file ISO Debian. Pilih partition scheme: GPT untuk UEFI atau MBR untuk Legacy BIOS.

üí° Tips: Gunakan USB 3.0 untuk proses yang lebih cepat
02
Konfigurasi BIOS

Boot dari USB

Restart komputer dan masuk ke BIOS/UEFI dengan menekan F2, F12, atau Del saat booting. Ubah boot priority ke USB Flash Drive, simpan pengaturan, dan restart.

Boot Priority: USB Drive ‚Üí HDD/SSD ‚Üí DVD
03
Pilihan Installer

Pilih Mode Instalasi

Pilih "Graphical Install" untuk instalasi dengan antarmuka grafis yang lebih mudah dan user-friendly, atau pilih "Install" untuk mode teks klasik.

Language: English / Indonesia
Location: Indonesia
Keyboard: American English

Konfigurasi Network

Masukkan hostname untuk komputer Anda (contoh: debian-laptop, debian-pc). Kosongkan domain name jika tidak diperlukan. Tunggu hingga konfigurasi jaringan selesai.

Setelah Instalasi

Konfigurasi dan tips untuk memaksimalkan Debian Anda

1

Update System

Selalu update system setelah instalasi untuk mendapatkan patch keamanan dan bug fixes terbaru.

bash 
sudo apt update
sudo apt upgrade -y
sudo apt dist-upgrade -y
2

Install Software Essentials

Install aplikasi yang sering digunakan untuk produktivitas dan multimedia.

bash 
sudo apt install firefox-esr \
  libreoffice vlc gimp \
  git curl wget htop
3

Install Codec Multimedia

Untuk memutar berbagai format video dan audio dengan sempurna.

bash 
sudo apt install \
  ffmpeg \
  gstreamer1.0-plugins-bad \
  gstreamer1.0-plugins-ugly
4

Setup Firewall

Aktifkan firewall untuk meningkatkan keamanan sistem Anda.

bash 
sudo apt install ufw
sudo ufw enable
sudo ufw status
